# casino纯娱乐性质小游戏方案

**背景：**我是一个Java后端开发从业者，有基于Java后端开发技术栈的开发经验。但是对于flutter开发毫无经验。

**想法：**我想制作一个纯娱乐性质的赌博游戏软件，可以在安卓手机和iphone上运行，iphone上的运行方案我准备仅在testflight上线，因为这是一个非营利、仅供身边朋友娱乐使用的项目。

## 初步计划：

1. 第一个版本只准备包含一个小游戏：21点，即BlackJack，后续可能继续添加德州扑克等游戏。
2. 第一个版本只考虑给安卓用户体验，因为我没有Mac。
3. 技术我希望使用flutter开发，能在开发的时候运行查看项目效果。v1只准备制作成一个无服务器、仅在本地运行的软件；
4. 发布使用Android APK 打包；
5. 软件制作完成后，希望能够提供一个网页，以当前本地为服务器，启动后可以供朋友访问下载安卓手机软件安装包。
